SNAKE
Rules of the game
  1. Use these keys to direct the snake and pick the "apple":
    • Up
    • Down
    • Left
    • Right
  2. The game will be over if the snake hits the wall.
  3. The snake cannot go backward, only forward.
  4. The more "apples" you collect, the faster the snake moves.
  5. Restart the game when it is over.

* Click the game name to see some cool effects.
